# Demo data

This directory stores **current baseline execution fixtures** used by local API smoke paths and runtime examples.

Use these assets for:

- seeded workflow registry rows
- stable runtime snapshots used during local/manual exercises
- sample payload fixtures for compile and flow-start flows

It is also the folder for small target/reference samples when you want to keep a test input next to its observed behavior.

If a dataset is intended as controller/Phase 7-only material, include it in the path and note it explicitly in the filename or companion doc.
