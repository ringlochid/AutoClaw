# 01 — Phase 1: Kernel and Data Model

## Goal

Establish deterministic source registries and compiled runtime truth.

## In scope

- role/policy/workflow/skill definitions and versions
- compiled plan generation with hash and snapshot
- basic runtime persistence and task execution bootstrap

## Outcome

- definitions and compilation are stable
- runtime can instantiate control nodes from compiled output

## Alignment note

This phase remains valid, and later phases refine the runtime tables and scheduler.
